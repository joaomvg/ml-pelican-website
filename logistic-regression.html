1. Logistic Regression

In the logistic regression algorithm, the probability in a binary classification problem, is calculated as

$$p(0|x)=\sigma\Big(\sum_i \omega_i x^i +b\Big)$$

where \(\sigma(x)\) is the sigmoid function

$$\sigma(z)=\frac{1}{1+e^{-z}}$$

We can represent this operation diagramatically as where \(x^i\) are the features of the datapoint \(x\). In the diagram, the node (circle) represents the composition of a linear operation followed by a non-linear function.

In the multiclass case, the probabilities have instead the form

$$p(y^k|x)=\frac{e^{z^k(x)}}{Z(x)}$$

where \(z^k=-\sum_i\omega^k_i x^i-b^k\) are the logits and \(Z(x)=\sum_l e^{z^l(x)}\) is a normalization. Diagrammatically this has the form

The function

$$f(w)^k=\frac{w^k}{\sum_l w^l}$$

is known as softmax. Since it is invariant under \(w\rightarrow \lambda w\) for non-zero \(\lambda\), we can fix a logit to zero. Take for example \(z^0=0\). This implies that \(\omega^0_i=0\) and \(b^0=0\).

Given a dataset \(S=\{(\vec{x}_0,y_0),(\vec{x}_1,y_1),\ldots (\vec{x}_N,y_N)\}\) we determine the parameters \(\omega\) and \(b\) using maximum-likelihood estimation, that is, we minimize the loss function

$$\begin{aligned}\mathcal{L}=&-\frac{1}{N}\sum_{i=1}^N \ln p(y^i|\vec{x}_i)\\ =&\frac{1}{N}\sum_i \omega^i_jx^j+b^i+\ln Z(\vec{x}_i)\end{aligned}$$

One of the main advantages of using the logistic function is that it makes the loss function convex, which allows us to apply more robust optimization algorithms like the Newton-Raphson method. To see this, lets for simplicity define \(\omega^k\equiv(\omega^k_i,b^k)\) and \(x\equiv(x^i,1)\). Calculating the derivatives of the loss function

$$\frac{\partial \mathcal{L}}{\partial \omega^{\mu}_{\nu}}=\langle \delta^k_{\mu}x^{\nu}\rangle_{y^k,x}-\langle x^{\nu}p_{\mu}(x)\rangle_{x}$$

where \(p_{\mu}(x)\) is the probability, \(\delta\) is the Kroenecker delta function; we are taking sample averages. And the second derivatives

$$\frac{\partial^2 \mathcal{L}}{\partial \omega^{\mu}_{\nu} \partial \omega^{\alpha}_{\beta}}=\langle x^{\nu}x^{\beta}\delta_{\mu\alpha}p_{\mu}(x) \rangle_x -\langle x^{\nu}x^{\beta}p_{\mu}(x)p_{\alpha}(x) \rangle_x$$

To show this is a convex optimization problem, build the quadratic polynomial at a particular point \(x\),

$$\begin{aligned}&\sum_{\mu,\nu,\alpha,\beta}x^{\nu}x^{\beta}\delta_{\mu\alpha}p_{\mu}(x)\omega^{\mu}_{\nu}\omega^{\alpha}_{\beta}-x^{\nu}x^{\beta}p_{\mu}(x)p_{\alpha}(x)\omega^{\mu}_{\nu}\omega^{\alpha}_{\beta}=\\ &\sum_{\mu} p_{\mu}(x)(\omega^{\mu}_{\nu}x^{\nu})^2-\Big(\sum_{\mu}p_{\mu}(x)\omega^{\mu}_{\nu}x^{\nu}\Big)^2=\langle \omega^2\rangle-\langle \omega\rangle^2\geq 0\end{aligned}$$

This result implies that

$$\sum_{\mu,\nu,\alpha,\beta}\frac{\partial^2 \mathcal{L}}{\partial \omega^{\mu}_{\nu} \partial \omega^{\alpha}_{\beta}} \omega^{\mu}_{\nu}\omega^{\alpha}_{\beta}\geq 0$$
